# Evaluación numérica de concentración de polifenoles en vino

## Descripción
Este proyecto consiste en la modelación numérica del proceso de concentración de polifenoles desde vino tinto mediante contactores de membrana.

El modelo considera transferencia de masa entre una solución de vino (12% etanol) y una solución de CaCl₂ al 20%, basado en ecuaciones de transporte y propiedades fisicoquímicas.

## Contenido
- `modelo.ipynb`: Implementación del modelo en Python

## Tecnologías
- Python
- NumPy
- SciPy
- Matplotlib

## Uso
Abrir el notebook en Jupyter y ejecutar las celdas en orden.
